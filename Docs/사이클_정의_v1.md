**조용/요란 판정 (레짐 경계 스위치)** — 사전 판별은 원리적 불가. **모든 개방 후 Qual 5장** 수행(소정비 포함), **직전 유효 스냅샷**과 비교해 판정 (유효 스냅샷 = 조용 판정 시 그 Qual로 갱신된 것 / 요란 이후엔 Phase 2 확정본 — R5):
- **판정식 [F12 백테스트 통과]**: 건강 센서 8종(C12 제외 — 제어 기준값이지 건강 신호 아님)의 σ-갭, **기준 산포 = 직전 사이클 전체 wafer 산포**, 임계 **3σ**. Qual 4지표(AE·σ-갭·Nelson·C65) 종합으로 판정하되 — **운영 실효 주력은 σ-갭 + AE 두 축** (둘 다 라벨 불요: C17 이동은 σ-갭이, 새 레짐 파형은 옛 레짐을 학습한 AE가 잡음). **C65 지표는 예측값(Proxy Metrology — Qual은 WT 미경유라 실측 없음)이라 과도기엔 기여 없음** (옛 모델의 예측은 낮게 나와 태연히 정상으로 보임). ⚠ F12 백테스트의 "C65 FAIL"은 실측 기준이었음 — 운영과 다름을 명시
- 산포 계산 시 **과도 구간(A7 제외 구간과 동일)은 제외** — 직전 사이클이 요란 레짐이면 과도가 산포를 부풀려 다음 판정이 둔감해짐 (R1)
- **부트스트랩**: 운영/시뮬 개시 시점의 초기 스냅샷·기준 산포는 **seg1 실측으로 시딩** (B3-1 초기 실력치 테이블과 동일 소스 — R3)
- **스냅샷 리베이스 (R5)**: 조용 판정 시 그 Qual로 스냅샷 즉시 갱신 / 요란 판정 시엔 **Phase 2 정식 재산정 때 새 레짐 스냅샷·기준 산포로 확정** — 리베이스 없으면 다음 소정비가 옛 레짐 스냅샷과 비교돼 **영구 '요란' 오판 루프**(bias 차단 연쇄). 장기 점진 표류 감시는 이 축이 아니라 SPC 추세 룰 + 리캘리 누적 상한(A3) 소관. *(검증 상태 주의: "조용" 판정 사례는 F12 백테스트에서 2건 확인(0.82σ·2.87σ) — 단 이는 "개방 직후 5장" 조건의 근사 표본이며, 정확한 조용-개방 전/후 쌍 실측은 없음(한계 ⑤). 조용 경로의 최종 검증은 시뮬 리허설에서)*
- **요란 ≠ 불량**: 요란은 "레짐 전환 감지"이지 불량 판정이 아님 (개선 방향 전환일 수도) — 후속은 자동 재정비가 아니라 **엔지니어 분기(S7: 복귀+가한계 / 추가 Qual / 재정비)**
- **대응은 유형이 아니라 판정에 건다 (원칙)**: 시스템의 모든 후속 로직(가한계·bias 리셋·리베이스·A7·TTTM 유예)은 작업 유형(소정비/대PM)이 아니라 **판정 결과(조용/요란)**에만 걸린다. 유형은 정비 기록·시뮬 생성·Evidence/사례/Scorecard의 **맥락 데이터**로만 사용. 따라서 "대PM인데 조용"(강도 분포의 ~70%)이면 소정비처럼 취급하는 것이 **올바른 동작** — 편차가 없는데 가한계를 돌릴 이유가 없음. "대PM임을 시스템이 어떻게 아는가"라는 질문 자체가 불필요해짐 (정비 주체는 기록으로 알고, 시스템은 알 필요 없음)

조건:   고정 표준 조건 (R7): 표준 레시피 C6_0 · 배치 영향 배제 ·
        센서 트레이스 = "현재 챔버 상태(레짐 운전점+오프셋)의 정착 구간 리샘플"
        → 무(無)에서 창조가 아니라 조건 통제된 리샘플 — 실측 앵커 유지
처리:   일반 wafer와 동일 파이프라인 (Qual 전용 코드 경로 없음 — 기획서 원칙)
        · B: 센서 σ-갭(스냅샷 대비) + Nelson  · A: AE score + C65 예측(proxy)
취합:   A5-3이 4지표 통합 → quals 테이블(PENDING) → S7 → 엔지니어 판정
후속:   조용 → qual_snapshots 갱신(R5) / 요란 → S7 분기 → 복귀 시 가한계 진입
실측:   Qual wafer는 fdc.actual 미발행 (WT 미경유 — 현실 정합)
```
- 요란 → AE 재학습(CT②) + 가한계 모드 진입 + rolling bias 리셋 / 조용 → 복귀 (bias·라벨 연속)

## 4. 운영 로직 정합

```
D+0     개방 (C33 리셋, pm_count+1) — 소정비든 대PM이든:
