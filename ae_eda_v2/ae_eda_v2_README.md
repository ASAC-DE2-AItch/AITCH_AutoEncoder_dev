## 노트북 목적
`merged_data.csv` ① 무결성 게이트 + 병합중복 정제, ② AE 입력 설계 파라미터 5종 확정. 데이터 사전 결론 인용, 병합본 성립만 스팟 체크.

## v1 대비 (왜 v2인가)
v1 실행에서 §5-1 게이트가 **병합 손상**을 검출(중복 1,456행). 진단 후 대응 3종 반영:
1. **dedup `(C64,C7,C46)` 1,456행 제거** 를 표준 전처리에 추가 (시간순 첫값)
2. **C41=C10−C39 판정 반올림 허용**(round, ≥99%)
3. **C65 wafer상수 위반 378장은 WARN**(AE 입력금지라 무영향, A트랙·PM 통보)

## 사용법
1. 의존성: 루트 `requirements.txt` (`pip install -r` → `pip freeze > requirements.lock.txt`). EDA 실필수 numpy·pandas·pyarrow.
2. `AE/ae_eda_v2/`에 두고 로컬 venv 커널로 위→아래 실행. v1이 만든 `merged_data.parquet` 재사용(원본, 중복포함) — dedup은 로딩 후 메모리.
3. 재현: seed=42. 게이트 3등급 PASS/★FAIL(불가침10 중단)/WARN(관찰·통보).
